## Project Overview

Build a native mobile app called **bushbeing** using Expo (React Native). The app is an indoor plant management tool.

**Tagline:** Nurture your indoor garden with intelligent plant care

## PRODUCT REQUIREMENTS

### Screens & Navigation (Bottom Tab Navigation)
1.  **Auth (Login/Register)**: Multi-step registration with OTP, login, forgot/change password.
2.  **My Plants (Home)**: Main dashboard with a grid of user's plants, filterable by room. FAB to add a new plant.
3.  **Add Plant**: Multi-step flow to upload plant photos, get AI-based identification (PlantNet), and save the new plant.
4.  **Plant Detail**: View plant details, care info, quick actions (water, edit, delete), and a health journal.
5.  **Hydrate**: A screen to track watering schedules, view today's progress, and log watering for past/present dates.
6.  **Nurture**: A dashboard for garden-wide stats and plant health cards with care tips.
7.  **Devices**: A Coming Soon placeholder screen for a future smart device integration.

### Feature Enhancements Requested by User:
1.  **Room Management**: Add, remove, or edit rooms from the My Plants screen.
2.  **Plant Edit Flow**: Allow changing a plant's room from a list of available rooms in the edit menu.
3.  **Watering Feedback**: Show success/undo messages and play a sound when a plant is watered.
4.  **Audio on Water**: Play a specific sound file () when watering.
5.  **Change Password**: Implement a functional Change Password option in a menu.
6.  **Drawer Menu**: Implement a side drawer menu on the main screen that can be dismissed by clicking outside.
7.  **UI Adjustments**: Move bottom tabs up for better screen fit and center-align room name text.
8.  **Auth UI/UX**: Add a repeat password field on registration, add input validation, and fix the password masking toggle.
9.  **UI Polish**: Improve spacing on dashboard top bars and redesign the Open Camera and Upload Picture buttons.
10. **Push Notifications**: Send push notifications for watering reminders.
11. **Error Notifications**: Display inline error banners on the Sign In and Register screens for unsuccessful attempts, instead of using system alerts.

**User's preferred language**: English
